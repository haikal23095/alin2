Pengertian Matrix:#

Matrix merupakan suatu kumpulan bilangan sejumlah m x n yang disusun menjadi m baris dan n kolom yang dapat dioperasikan (penjumlahan,pengurangan,dan perkalian). Matrix juga bisa menukar antara nilai baris dan kolomnya yang disebut Transpose Matrix.

Membuat Matrix:#

Cara membuat Matrix yaitu kita bisa menulisakan kumpulan angka dalam kurung siku. Contoh Matrix dibawah ini, ada matrix_A (3x3), matrix_B (3x3), matrix_C (3x2), dan matrix_D (3x3)

\( \begin{align*} A= \begin{bmatrix} 1&3&9\\ 3&-1&6\\ 9&4&1 \end{bmatrix} \end{align*} \)

\( \begin{align*} B= \begin{bmatrix} 1&7&5\\ 2&4&5\\ 1&-1&4 \end{bmatrix} \end{align*} \)

\( \begin{align*} C= \begin{bmatrix} 3&2\\ -1&2\\ 3&-5 \end{bmatrix} \end{align*} \)

\( \begin{align*} D= \begin{bmatrix} 2&4&8\\ 2&-1&5\\ -8&5&2 \end{bmatrix} \end{align*} \)

Membuat Matrix di Python:#

Berikut adalah cara Membuat Matrix di Python:

  1. Import Library numpy simpan dalam variabel np

  2. Deklarasikan variabel untuk menyimpan matrix (disini saya menggunakan matrix dengan vaariabel A (3x3), B (3x3), C (3x2), dan D (3x3))

  3. Lalu panggil variabel np diikuti dengan fungsi .array untuk membuat rangkaian matrix

  4. Lalu untuk menampilkan matrix kita gunakan fungsi print

  5. Untuk menampilkan ukuran matrix bisa kita gunakan fungsi .shape, untuk contohnya kalian bisa lihat potongan code berikut

import numpy as np

# Matrix
A = np.array([[1,3,9],[3,-2,6],[9,4,1]]) # matrix 3x3
B = np.array([[1,7,5],[2,4,5],[1,-1,4]]) # matrix 3x3
C = np.array([[3,2],[-1,2],[3,5]]) # matrix 3x2
D = np.array([[2,4,8],[2,-1,5],[-8,5,2]]) # matrix 3x3

print(f"Matrix A : \n{A}\n ukurannya matrixnya yaitu {A.shape}")
print(f"Matrix B : \n{B}\n ukurannya matrixnya yaitu {B.shape}")
print(f"Matrix C : \n{C}\n ukurannya matrixnya yaitu {C.shape}")
print(f"Matrix D : \n{D}\n ukurannya matrixnya yaitu {D.shape}")

Operasi Matrix#

Matrix juga bisa kita lakukan operasi yaitu Penjumlahan, Perkalian, dan Pengurangan. Untuk penjumlahan dan pengurangan hanya bisa dilakukan oleh matrix yang memiliki ordo yang sama. Sedangkan perkalian antar matrix bisa dilakukan jika baris pada matrix_a sama dengan jumlah kolom pada matrix_b.

Operasi Penjumlahan Matrix#

Operasi penjumlahan matrix adalah penjumlahan antara nilai nilai yang ada pada dua buah Matrix. Penjumlahan dapat dioperasikan secara langsung antara baris dan kolom yang sama.

Seperti contoh berikut yang diimplementasikan bentuk dalam code berikut:

# Matrix A+B
print(f"Matrix A : \n{A}")
print(f"Matrix B : \n{B}")
E = (A+B)
print(f"Matrix E = Matrix A + Matrix B:\n{E}")
print(f"Ukuran Matrix E = {E.shape}")

Matrix A : 
[[ 1  3  9]
 [ 3 -2  6]
 [ 9  4  1]]
Matrix B : 
[[ 1  7  5]
 [ 2  4  5]
 [ 1 -1  4]]
Matrix E = Matrix A + Matrix B:
[[ 2 10 14]
 [ 5  2 11]
 [10  3  5]]
Ukuran Matrix E = (3, 3)

Operasi Perkalian Matrix#

Operasi perkalian matrix ada dua macam

  • perkalian Matrix dengan sebuah nilai tertentu. Pada perkalian ini kita bisa operasikan langsung antara nilai nilai yang ada di dalam matrix dengan sebuah nilai tertentu.

  • perkalian antara dua buah Matrix. perkalian antara dua buah matrix dapa disimbolkan dengan matrix_a @ matrix_b atau matrix_a . matrix_b. Syarat untuk mengkalikan antara dua buah matrix yaitu jumlah kolom pada matrix_a harus sama dengan jumlah baris pada matrix_b.

# Perkalian Matrix A dengan angka 2 yang menghasilkan matrix F
print(f"Matrix A = \n{A}")
F = 2*(A)
print(f"Perkalian 2 * Matrix A =\n{F}")
print(f"Ukuran Matrix = {F.shape}")

Tranpose Matrix#

Anggap Baris = m, Kolom = n

=> C = A_trans @ A

=> (n x m) @ (m x n) = n * n

contoh: [2 x 3] @ [3 x 2] = [2 x 2]

=> C = A @ A_trans

=> (m x n) @ (n x m) = m x m

contoh: [3 x 2] @ [2 x 3] = [3 x 3]

# Contoh dengan Martrix 3x3

A_tran = A.transpose()
print('Matrik A sebelum di transpose=\n', A)
print()
print('Matrik A setelah di transpose=\n',A_tran)
print('Ordo A_transpose adalah',A_tran.shape)
print()
Z = A_tran @ A
print('Z = A_transpose x A =\n',Z)
print('Ordo Z adalah',Z.shape)
print()
P=A @ A_tran
print('P = A_transpose x A =\n',P)
print('Ordo P adalah',P.shape)

Program untuk mengkalikan Matrix A dengan Matrix B Secara Manual tidak menggunakan ‘@’#

Dalam program ini saya menggunkan perulangan for untuk mendapatkan hasil yang sesuai yaitu dengan membuat perulangan dalam perulangan (nested for).Selanjutnya mengkalikan antara value-valuenya dan menampungnya dalam variabel result. Contohnya dalam gambar seperti ini:

Matrix_1 = [[1,3,9],
            [3,-2,6],
            [9,4,1]]

Matrix_2 = [[1,7,5],
            [2,4,5],
            [1,-1,4]]

print(f" Matrix_1 = {Matrix_1}")
print()
print(f" Matrix_2 = {Matrix_2}")

def multiply_matrix(matrix1,matrix2):
  result = [[0,0,0],[0,0,0],[0,0,0]] # Menginisialisasikan dulu matrix dengan semua value 0

  for i in range(len(Matrix_1)):
    for j in range(len(Matrix_2[0])):
      for k in range(len(Matrix_2)):
        result[i][j] += Matrix_1[i][k] * Matrix_2[k][j]
  return result

hasil = multiply_matrix(Matrix_1,Matrix_2)
print()
print(f"Hasil perkalian Matrix_1 dan Matrix 2 = {hasil}")
print()
print(f"Pembuktian untuk Matrix A @ B =\n{A @ B}")
